## 为什么要使用hexo？
hexo简单好用



## 下载node
作者这里就只提供一下网站了，安装很简单，[node](https://nodejs.org/en)


## 下载git
不会git，建议先学习，这里推荐下载git的比较快的网站，[git](https://registry.npmmirror.com/binary.html?path=git-for-windows/)

## 查看版本
```shell
$ node -v
```
## 下载Hexo
```shell
$ npm install -g hexo-cli
```
## 安装Hexo
在你的某个文件夹下，执行：
```shell
$ hexo init
```
